# LogiMetrics Pro - Fleet Management Dashboard

O **LogiMetrics Pro** é um dashboard interativo para gestão de frotas logística em tempo real. O sistema permite monitorar o status de equipamentos, calcular custos de viagem através de um mapa interativo e gerenciar o inventário diretamente no banco de dados.

## Tecnologias Utilizadas

* **Backend:** Java 17, Spring Boot 3, Spring Data JPA.
* **Banco de Dados:** PostgreSQL (Rodando em container Docker).
* **Frontend:** HTML5, CSS3 (Neon Design System), JavaScript (Fetch API).
* **Mapas:** Leaflet.js com integração de geolocalização.
* **Infraestrutura:** Docker e Docker Compose.

##  Funcionalidades

- **Dashboard Real-time:** Visualização de máquinas Ativas, em Manutenção ou Paradas.
- **Gestão de Inventário (CRUD):** Adição de equipamentos via catálogo, alteração de status e remoção.
- **Cálculo de Logística:** Seleção de dois pontos no mapa para estimativa de custo baseada no KM da máquina escolhida.
- **Filtros Dinâmicos:** Segmentação da frota por estado de operação.
